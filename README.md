# **LeetCode-Thinking**

本仓库主要用来记录我在LeetCode上的解题思路，后期可能会添加牛客网、codility、Lintcode等网站的解题思路。

本仓库参照LeetCode分类，目前分为4大类（数据库、算法、Shell脚本、多线程）。每大类又按难度分为简单、中等、困难。
